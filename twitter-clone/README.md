# Twitter Clone

Twitter clone is a based-rails app that allows you send post and follows people.

## Requirements

### Gem
* gem 'devise'
* gem 'devise-bootstrap-views', '~> 1.0'
* gem 'will_paginate'
* gem 'gravatar_image_tag'
* Rails 6.1
